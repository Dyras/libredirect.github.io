LibRedirect

An extension that redirects popular sites to alternative privacy-friendly frontends and backends

YouTube => Invidious, YouTube

Twitter => Nitter

Instagram => Bibliogram

Reddit => libreddit, Teddit, Snew

Google => SearX, DuckDuckGo, Startpage, Ecosia, Qwant, Mojeek, Presearch, Whoogle

Google Translate => SimplyTranslate

Google Maps => OpenStreetMap

Wikipedia => Wikiless

Medium => Scribe

It's a free and open source software licensed under the GPL-3.0. If you have any questions, feature requests, or found a bug, feel free to open an issue on GitHub.
